# Advanta365

Premium Microsoft 365 adoption and rollout website built with Next.js App Router.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Project structure

- `app/page.jsx` - Advanta365 homepage UI and browser interactions
- `app/layout.jsx` - site metadata and root layout
- `app/globals.css` - global responsive styling and motion design
- `next.config.mjs` - Next.js configuration
